# untodo changelog

## v4.0.0 — The Big 40 (Round 40)
*March 14, 2026*

Full app review, final polish, and milestone release. 40 rounds of overnight shipping.

### Core
- Task management: create, edit, delete, complete, reorder
- Subtasks with progress tracking
- Task notes and estimated time
- Priority levels (high, medium, low) with visual indicators
- Categories: work, personal, health, learning, finance, creative
- Recurring tasks (daily, weekly, custom days)
- Carry-over of incomplete tasks to next day
- Undo system for accidental deletes
- Configurable day reset hour (0-12)
- Swipe gestures: right to complete, left to delete

### Productivity
- Pomodoro timer with 7 presets (Spark, Classic, Coder 50, DeskTime, Deep Block, Flowtime, Custom)
- Flowtime: open-ended focus sessions
- Per-task time tracking (start/stop)
- Focus mode: distraction-free single-task view
- Daily score with letter grades (A+ through F)
- Quick actions: long-press menu for fast operations

### Progress & Analytics
- Daily completion ring with percentage
- 7-day streak counter with milestones
- Weekly bar chart
- 91-day contribution heatmap (GitHub-style)
- Deep stats: avg tasks/day, peak hour, longest streak, category breakdown, most productive day
- Daily summary and weekly review
- Morning brief with daily overview
- Task insights: contextual tips based on your patterns
- Share progress as image cards

### Achievements
- 10 unlockable achievements (First Task, Getting Started, Week Warrior, etc.)
- Toast notifications on unlock
- Achievements grid in progress screen

### Wallpaper Engine
- Generate phone wallpapers from your task data
- Multiple visual styles and color themes
- Streak display, daily quotes, dot grid
- Save to gallery
- Phone frame preview

### Notifications
- Morning motivation (10am)
- Afternoon check-in (3pm)
- Evening reminder (9pm)
- Weekly summary (Sunday 7pm)
- Persistent progress notification with live updates

### Sync & Data
- Firebase Firestore real-time sync
- Offline queue with automatic retry (up to 200 items)
- Data export: JSON and CSV
- Data sanitization and migration safety
- Error boundaries for crash recovery

### Silicon Integration
- AI assistant pairing via 6-character code
- Command bridge: add tasks, get tasks, nudge, write summaries
- 5-second polling for live connection

### UX
- Animated 3-step onboarding
- Skeleton loading states
- Pull-to-refresh
- Haptic feedback throughout
- Gesture tutorial for new users
- Calendar date picker
- Time-aware greetings
- Daily rotating quotes (80+ stoic/memento mori)
- Dark theme (premium minimal aesthetic, #0A0A0A)
- Confetti celebrations on task completion
